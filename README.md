## Problem :-

Applying for multiple internships across different platforms like Linkedin , Naukri , etc ., can get messy. Tracking deadlines and following up is often a manual tedious task .

## Solution :-

I built a full stack web application that not only stores my application history but also automates the remainder process!

## Key Features :-

* **Custom Dashboard** :- Built with Flask and Bootstrap 5 for a clean , modern UI .
* **Database Management** :- Uses SQLite and SQLAlchemy to securely store application details .
* **Automation Engine** :- Integrated with n8n ( Low-Code automation ) to fetch data via HTTP requests .
* **Smart Reminders** :- An automated workflow that triggers personalized email alerts using an AI agent logic .
* **Scheduled Tasks** :- Uses Flask - APScheduler to check for upcoming deadlines every morning .

## Tech Stack :-

* **Backend** :- Python , Flask .
* **Frontend** :- HTML 5 , CSS , Bootstrap 5 .
* **Database** :- SQLite .
* **Automation** :- n8n .
* **Version Control** :- Github .
* **AI Tool For Creating the Project** :- Gemini .
